# UIDAI Data Hackathon 2026 – Lifecycle Analysis of Aadhaar Activity

This repository contains supporting analysis notebooks and visual assets for the UIDAI Data Hackathon 2026 submission.

## Overview
The project analyses Aadhaar enrolment, demographic updates, and biometric updates as a lifecycle system to identify operational and planning signals at the state–month level.

## Contents
- `notebooks/` – Data cleaning, aggregation, and analysis notebooks
- `visuals/` – Final charts used in the submission report
- `report/` – Submission-ready PDF (for reference)

## Notes
- All analysis uses only officially provided, aggregated, and anonymised UIDAI datasets.
- This repository is intended for transparency and reproducibility.
- The primary evaluation artifact is the submitted PDF report.
